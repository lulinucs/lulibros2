# Changelog - Lulibros 2.0

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2024-10-17

### Adicionado
- Sistema completo de gestão de banca de livros
- Autenticação JWT
- API REST em PHP com PDO
- Frontend React/TypeScript com Vite
- Sistema de upload de CSV para produtos
- Gestão de preços e estoque
- Sistema de vendas
- Gestão de clientes
- Controle financeiro (caixa)
- Relatórios e estatísticas
- Interface responsiva com Tailwind CSS
- Sistema de filtros e busca
- Paginação
- Configuração de ambientes (dev/prod)
- Documentação completa
- Scripts de setup automatizado

### Funcionalidades Principais
- **Produtos**: Cadastro, edição, consulta e upload em lote
- **Vendas**: Processo completo de venda com controle de estoque
- **Clientes**: Cadastro e gestão de clientes
- **Financeiro**: Controle de caixa e movimentações
- **Relatórios**: Estatísticas e relatórios de vendas
- **Usuários**: Sistema de autenticação e autorização

### Tecnologias
- **Backend**: PHP 8.0+, MySQL, PDO, JWT
- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS
- **Banco**: MySQL com charset UTF8MB4
- **Autenticação**: JWT com middleware
- **Upload**: Suporte a CSV com validação

### Configuração
- Ambientes separados (desenvolvimento/produção)
- Variáveis de ambiente configuráveis
- Scripts de setup para Windows e Linux
- Documentação de instalação e deploy
